How to run the Algorithms

1. Install anaconda from https://docs.anaconda.com/free/anaconda/install/index.html
2. Follow installation instructions for osmnx https://osmnx.readthedocs.io/en/stable/#installation
3. Run the algorithms on the osmnx kernel 
4. VScode and Jupyter notebook are required to run the code
5. There may be dependencies or packages that may need to be installed. Follow error prompts to install the required packages.
